# NIFTY Option Pricing Backend (FastAPI + Supabase)

Backend for real-time option pricing, Greeks, IV calculation, and visualization (heatmap, expiry graph) for NIFTY 50.

## Tech
- FastAPI, Python 3.11
- Supabase (PostgreSQL)
- NumPy, SciPy, Pandas
- Matplotlib/Seaborn
- APScheduler
